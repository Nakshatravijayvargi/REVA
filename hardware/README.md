| Component | Quantity |
|-----------|----------|
| Arduino Uno | 1 |
| HC-SR04 Ultrasonic Sensor | 1 |
| SG90 Servo Motor | 1 |
| Breadboard | 1 |
| Jumper Wires | Several |
| USB Cable | 1 |

---

## Working

1. Ultrasonic sensor detects an object within **25 cm**.
2. Arduino processes the measured distance.
3. Servo motor rotates to open the lid.
4. After a short delay, the lid closes automatically.
5. The dashboard demonstrates monitoring of the smart waste management system.
